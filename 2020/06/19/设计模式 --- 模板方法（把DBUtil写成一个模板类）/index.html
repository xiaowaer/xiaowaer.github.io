模板模式简介

定义：定义一个模板结构，将具体内容延迟到子类去实现。

主要作用：在不改变模板结构的前提下在子类中重新定义模板中的内容。

优点：

  1. 提高代码复用性
      将相同部分的代码放在抽象的父类中，而将不同的代码放入不同的子类中 实现了反向控制
       通过一个父类调用其子类的操作，通过对子类的具体实现扩展不同的行为，实现了反向控制 & 符合“开闭原则”
    注意：模板是基于继承实现的。

抽象模板类

public abstract class Game {
   abstract void initialize();
   abstract void startPlay();
   abstract void endPlay();

   //模板
   public final void play(){
      //初始化游戏
      initialize();
      //开始游戏
      startPlay();
      //结束游戏
      endPlay();
   }
}

具体模板类

public class Cricket extends Game {
   @Override
   void endPlay() {
      System.out.println("Cricket Game Finished!");
   }

   @Override
   void initialize() {
      System.out.println("Cricket Game Initialized! Start playing.");
   }
   @Override
   void startPlay() {
      System.out.println("Cricket Game Started. Enjoy the game!");
   }
}

DBUtil模板类的实现

简介:使用模板方法对JDBC常用的连接,关闭,查询,修改做出DBUtil模板工具类，并把项目中的所有DaoImpl类继承它，作为模板实现类来使用。

DBUtil模板类

 import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.PreparedStatement;
import java.sql.ResultSet;
import java.sql.SQLException;
import java.util.ArrayList;
import java.util.List;

public abstract class DBUtil<T> {

    Connection conn=null;
    PreparedStatement pst =null;
    ResultSet rs = null;

    public Connection  getConn() {
        try {
            Class.forName("com.mysql.jdbc.Driver");
            conn = DriverManager.getConnection("jdbc:mysql://127.0.0.1:3306/day0617?characterEncoding=utf-8", "root",
                    "lijunjie123");
        } catch (ClassNotFoundException | SQLException e) {
            // TODO Auto-generated catch block
            e.printStackTrace();
        }

        return conn;
    }

    public void closeconn(Connection conn,PreparedStatement pst,ResultSet rs) {
        try {
            if(rs!=null)
                rs.close();
            if(pst!=null)
                pst.close();
            if(conn!=null)
                conn.close();
        } catch (Exception e) {
            // TODO: handle exception
        }
    }
    //通用模板
    public boolean update(String sql,Object...objects) {
        try {
            pst=getConn().prepareStatement(sql);
            for (int i=0;i<objects.length;i++) {
                pst.setObject(i+1, objects[i]);
            }
            int row=pst.executeUpdate();
            if(row>0)
                return true;
                    return false;
        } catch (SQLException e) {
            // TODO Auto-generated catch block
            e.printStackTrace();
        }finally {
            closeconn(conn, pst, rs);
        }
        return false;

    }
    //查询方法
    public List<T> query(String sql,Object...objects) {
        List<T> list =new ArrayList<>();
        try {
            pst=getConn().prepareStatement(sql);
            for (int i=0;i<objects.length;i++) {
                pst.setObject(i+1,objects[i]);
            }
            rs=pst.executeQuery();
            while (rs.next()) {
            list.add(getEntity(rs));

            }
            return list;
        } catch (Exception e) {
            // TODO Auto-generated catch block
            e.printStackTrace();
        }finally {
            closeconn(conn, pst, rs);
        }
        return list;

    }

    public abstract T getEntity(ResultSet rs) throws Exception;

}

为了实现在查询时，将结果集的参数放入到不同的实体类中，并添加到list中作为返回结果，把结果集装入实体类的过程抽象成一个泛型方法，可在子类中实现结果集装入把不同的实例。
