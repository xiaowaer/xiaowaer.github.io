分类二：数据结构相关

数据类型：

  1. bool：布尔类型，C++中的基本数据结构。
  2. char：C++中的基本数据结构。
  3. double：(双精度）类型，C++ 中的基本数据结构，以双精度形式存储一个浮点数。
  4. float（浮点数）：C++ 中的基本数据结构，精度小于 double。
  5. int:（整型，integer），C++中的基本数据结构，用于表示整数，精度小于 long。
  6. long:（长整型），C++中的基本数据结构，用于表示长整数。
  7. short:（短整型），C++中的基本数据结构，用于表示整数，精度小于 int。
    1. struct（结构）类型，类似于 class 关键字，与 C 语言兼容（class 关键字是不与 C 语言兼容的），可以实现面向对象程序设计。
    2. enum（枚举）类型：给出一系列固定的值，只能在这里面进行选择一个。
    3. union（联合），类似于 enum。不同的是 enum 实质上是 int 类型的，而 union 可以用于所有类型，并且其占用空间是随着实际类型大小变化的。
    4. wchar_t 是宽字符类型，每个 wchar_t 类型占 2 个字节，16 位宽。汉字的表示就要用到 wchar_t。
    5. 取值
  8. true：C++ 的基本数据结构 bool 类型的值之一。等同于 int 的非 0 值。
  9. false（假的）， bool类型的值之一。等同于 int 的 0 值。
    符号
    1. unsigned（无符号）: 表明该类型是无符号数，和 signed 相反。
    2. signed（有符号）:表明该类型是有符号数，和 unsigned 相反。数字类型（整型和浮点型）都可以用 signed 修饰。但默认就是 signed，所以一般不会显式使用。
      计算大小
      sizeof：由于 C++ 每种类型的大小都是由编译器自行决定的，为了增加可移植性，可以用 sizeof 运算符获得该数据类型占用的字节数。

循环跳出

  1. continue：因而它是”继续循环”之意，不是 break（跳出）。
  2. break：（中断、跳出），用在switch语句或者循环语句中。程序遇到 break 后，即跳过该程序段，继续后面的语句执行。

分类4：异常处理

try-catch： 异常捕获处理机制。
throw：抛出异常处理机制。

分类5：程序控制

return：（返回）用于在函数中返回值。程序在执行到 return 语句后立即返回，return 后面的语句无法执行到。
void（空的）：可以作为函数返回值，表明不返回任何数据；可以作为参数，表明没有参数传入（C++中不是必须的）；可以作为指针使用。

分类6：指针操作

new（新建）用于新建一个对象。new 运算符总是返回一个指针。由 new 创建。
delete（删除）释放程序动态申请的内存空间。delete 后面通常是一个指针或者数组 []，并且只能 delete 通过 new 关键字申请的指针，否则会发生段错误。
